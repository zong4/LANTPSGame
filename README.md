# 背景

太空漫游hh。

# 玩法

通过碰撞来让对方掉下去死亡（无血条），每人两条命，部分操作设置可在设置界面修改。

# 提示

特此声明，如果单人游玩会出现 **Game Over** 字样属正常现象，游戏目前没有设置单人模式，敬请期待。
